
# Habit Tracker

Track your habits everyday . 


## Run Locally

Clone the project

```bash
  git clone [https://link-to-project](https://github.com/Nisarg39/HabitTracker.git)
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
